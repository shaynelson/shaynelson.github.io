How To Make a Github Page

    Step 1: Make an account at Github
    Step 2: Create a repository
    Step 3: Name your repository
    Note: Be sure to keep ".github.io" after your name
    Step 4: Create a simple index.html file
    Step 5: Click to upload file or files
    Step 6: Drag index.html file to upload
    Step 7: Go to settings
    Step 8: Go to your Pages settings
    Step 9: Choose a Theme
    Note: Any theme will work. You will using it
    at this time
